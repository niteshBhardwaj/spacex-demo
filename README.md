# GraphQL React Application

A demo app for using GraphQl, Appollo Client with React and SpaceX API.

## Installation:

1. Clone this repo by running `git clone https://github.com/niteshBhardwaj/spacex-demo.git`
2. `yarn install`
3. `yarn start`

## Built With :zap:

1. React
2. Material UI
3. GraphQL
4. Appollo Client
5. Typescript
